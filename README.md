JENNY takes long-form whitepapers and breaks them into sharp, publishable X.com content — threads, standalone posts, and multi-day series.

Named after a very good golden retriever.

## Quickstart

```bash
npx skills add FnSK4R17s/JENNY
```

Summon JENNY:

```text
JENNY
```

Or point her at a whitepaper directly:

```text
JENNY, break down my whitepaper at whitepaper-output/ai-chips/whitepaper/ai-chips-whitepaper.md into X articles
```

## Works With BEARY

JENNY automatically discovers whitepapers produced by [BEARY](https://github.com/sally-jankovic/BEARY) by reading BEARY's configured output path. No extra configuration needed — just run BEARY first, then summon JENNY.

You can also point JENNY at any markdown whitepaper, regardless of how it was produced.
